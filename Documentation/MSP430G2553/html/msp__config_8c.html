void SetUART_115200_16MHz ( void  )

Set UART Function.

Configures the UART hardware to work at 115200 bauds & 8N1 mode with a clock at 16 MHz. This mode can be used only with a external usb-serial adapter connected to RXD & TXD pins. WARNING: Don't forget to remove the UART junpers from the Launcpad.

void SetUART_9600_16MHz ( void  )

Set UART Function.

Configures the UART hardware to work at 9600 bauds & 8N1 mode with a clock at 16 MHz. This mode is the recomended to work with the integrated usb-serial in Launchpad
